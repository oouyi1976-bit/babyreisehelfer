# Offizielle Amazon-Bild-Links für den Ratgeber

Stand: 19. Juli 2026

Für die untenstehenden Produktkarten liegt im Projekt kein SiteStripe- oder Product-Advertising-API-Bild-Link vor. Deshalb verwenden die Karten weiterhin eigene neutrale lokale Produktbilder. Es wurden keine Amazon-Bilder kopiert, heruntergeladen oder aus Seitenquelltexten übernommen.

## So werden offizielle Bilder ergänzt

1. Im Amazon-PartnerNet für das jeweilige Produkt einen **SiteStripe Bild-Link** oder **Text-und-Bild-Link** erzeugen, oder die Bild-URL über die offiziell freigeschaltete Product Advertising API beziehen.
2. Die freigegebene Bild-URL als `officialAmazonImageUrl` beim passenden Objekt in `src/data/travelProducts.ts` eintragen.
3. Keine Bilddatei lokal speichern, keinen Screenshot verwenden und keine URL aus Amazon-HTML kopieren.
4. Die Karte nimmt die offizielle URL automatisch vor dem lokalen Bild in Anspruch.
